Komponen Hardware NOVA

ESP32-S3 WROOM

Microcontroller utama dengan WiFi dan Bluetooth terintegrasi

  • 240 MHz dual-core
  • 512KB SRAM, 8MB PSRAM
  • 2.4 GHz Wi-Fi
  • Bluetooth 5.0

OLED 128x64

Layar OLED untuk menampilkan jam dan emoji

  • Resolusi 128x64
  • Brightness tinggi
  • I2C interface
  • Low power

MAX9814 Microphone

Microphone dengan amplifier untuk deteksi suara

  • Gain 50dB
  • Automatic gain control
  • Sensitivitas tinggi
  • Noise cancellation

DS3231 RTC

Real Time Clock untuk menjaga akurasi jam

  • Akurasi ±2ppm
  • Battery backup
  • Temp sensor
  • Alarm & timer

ESP32-S3

240MHz Dual-Core dengan AI Instructions

8MB PSRAM

Memori tambahan untuk buffer suara

Cara Install Library

1. Buka Arduino IDE
2. Tools → Manage Libraries
3. Cari "Adafruit SSD1306"
4. Klik Install
5. Ulangi untuk library lainnya
